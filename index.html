O que é HTML

(Voltar)

HTML é uma linguagem de marcação utilizada na construção de páginas na WEB. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo.

O que são Clients?

(Voltar)

É uma linguagem que é executada do lado do cliente, ou seja no computador do próprio usuário. O JS roda do lado do cliente é a unica linguagem que roda do lado do cliente e do servidor

O que são Servers?

(Voltar)

É uma linguagem que é usada no servidor, quando você cria essa aplicação ela vai rodar do lado do servidor não do cliente. Linguagens que roda do lado do servidor: PhP, Ruby, C#, JS, Java.

Como é uma estrutura básica de um HTML?

(Voltar)

O HTML consiste em 3 partes, a tag que abre <> e uma tag que fecha , o index é o padrão dizendo q é o começo, a página principal e o HTML consiste em duas partes, a cabeça(head) e o corpo(body) o que estiver entre o corpo vai aparecer para o usuário, o head são coisas que vai carregar antes de passar para o usuário e o body vai exibir para o usuário.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Esses dois tipos de listas são o padrão, a lista ordenada(ol) você não pode sair digitando dentro dela você precisa usar o li dentro dela para usar um item em baixo do outro, quando a lista é ordenada o navegador vai entender que tem uma ordem. listas não ordenadas(ul) é uma lista de itens, são marcadas por bullets(uma bolinha preta) e para escrever dentro dela você também usa o li.
